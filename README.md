It uses public scraping tools and yahoo finance (limited to 60 days back) so script can just run natively just inserty strategy ID and you are good to go. 

  Prerequisites (windows and mac):
  Install Node.js first: https://nodejs.org (download LTS version)
To verify it's installed:
  node --version

then wherever file located just run in terminal/command: node intraday-analyzer-standalone.js
